Package Structure
-----------------

``btorch`` is organized into three main functional areas:

- **btorch.models**: Spiking neuron models and network components
- **btorch.analysis**: Spike train and neural data analysis tools
- **btorch.connectome**: Connectome data handling and connectivity utilities
