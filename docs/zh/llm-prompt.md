# Translation Guidelines for Chinese

- Use formal, academic Chinese suitable for engineering documentation.
- Keep code blocks, Python identifiers, URLs, and math notation unchanged.
- Do not translate the following terms; use the specified Chinese where indicated:
  - Spiking Neural Network -> 脉冲神经网络
  - connectome -> 连接组
  - truncated BPTT -> 截断时间反向传播
  - torch.compile -> torch.compile
  - register_memory -> register_memory
  - neuron -> 神经元
  - surrogate gradient -> 替代梯度
  - ONNX -> ONNX
- Prefer concise technical expressions over literary flourishes.
